# Cohésion du recueil — Techniques pour recueils de nouvelles

## Techniques identifiées sur Station-Service Désert

1. **Détail-partagé** : un détail (calendrier 1987, 15 août entouré) planté dans R01 comme ambiance, repris dans R02 comme point nodal.
2. **Polyvalence du décor** : la même station vue comme menace (R01), tombeau (R02), refuge (R03).
3. **Temporalité flottante** : les 3 récits contemporains sans synchronisation.
4. **Cohésion par la lumière** : nuit noire (R01) → crépuscule gris (R02) → bougie + aube rose (R03).
5. **Variation temporelle d'un objet** : photo souvenir du passé (R02) vs photo vestige anticipé (R03).

**Principe** : les récits sont indépendants mais les échos créent une profondeur sans dépendance.

# Skill — Écriture hybride

> Mélange des genres : prose, poésie, théâtre, essai — dans un même texte, sans perdre le lecteur.
> Mis à jour le 02/06/2026 — Création initiale depuis le template.

## Principes

- **Le genre est un outil, pas un cadre** : chaque séquence peut adopter la forme qui sert le mieux son propos. Le roman n'est pas un bloc — il peut contenir des séquences poétiques, théâtrales, épistolaires.
- **Signalétique de transition** : tout changement de genre doit être signalé (blanc typographique, italique, changement de format). Le lecteur ne doit jamais être perdu.
- **Pacte de lecture local** : chaque séquence respecte les règles de son genre. Ce qui est acceptable en poésie (image visionnaire) ne l'est pas forcément dans la séquence en prose qui suit.
- **L'hybridation sert le sens** : changer de genre n'est pas décoratif — c'est une nécessité du récit. Si on peut écrire la même scène en prose, le faire. L'hybridation se justifie par ce qu'elle révèle que la prose seule ne peut pas dire.
- **Signature d'auteur** : l'hybridation générique est une marque de voix. Elle ne se force pas — elle vient quand le texte le demande.
- **La didascalie narrative montre, n'explique pas** : dans une séquence théâtrale, la didascalie décrit ce qui est visible (un geste, une lumière, un silence). Elle n'interprète jamais ce que le personnage ressent ni ce que le spectateur doit comprendre. Si le geste seul ne suffit pas à dire l'émotion, c'est le geste qu'il faut retravailler — pas la didascalie qu'il faut enrichir. Une didascalie qui dit « Elle sait que c'est fini » trahit le pacte théâtral ; une didascalie qui dit « Sa main reste en l'air, suspendue » laisse le silence faire le travail.

## Boîte à outils de l'écriture hybride

Techniques disponibles, jamais obligatoires — l'auteur pioche selon les besoins du texte. La rareté fait la puissance.

- **Séquence théâtrale dans le roman** : didascalies en italique, noms de personnages en capitales, répliques sans guillemets. Fonctionne pour les scènes de confrontation, les interrogatoires, les aveux. Une par roman maximum, ou l'effet se dilue.
- **Poème dans la prose** : un poème inséré dans un récit en prose crée une respiration — il dit par l'image ce que la prose ne peut pas dire par la narration. Le poème doit être justifié (un personnage qui écrit, une lettre, un graffiti).
- **Passage meta-textuel** : le narrateur sort un carnet et note dans le récit (« J'écris : », « Je note : »). Le texte du carnet en italique, plus dense, crée une mise en abyme. Doit révéler l'état du personnage, pas orner le récit.
- **Changement de registre typographique** : italique pour la pensée, capitales pour le cri, blanc pour le silence. Chaque variation typographique est un signal adressé au lecteur.
- **Blanc comme séparateur de genre** : un double saut de ligne (ou une page blanche) entre une séquence en prose et une séquence poétique. Le blanc prépare le lecteur au changement de régime.
- **Fusion des voix** : le même narrateur peut parler en prose dans un chapitre et en vers dans le suivant — à condition que la voix (le regard, les objets, le rythme) reste reconnaissable.
- **Didascalie sensible vs explicative** : une didascalie sensible montre le visible (« Sa main reste à quelques centimètres de son visage, suspendue. »). Une didascalie explicative dit l'invisible (« Il veut la toucher mais n'ose pas. »). Le théâtre pur interdit l'explicative. En hybridation roman/théâtre, l'explicative peut être tolérée dans les passages narratifs (le romanesque) mais jamais dans les séquences purement théâtrales (les didascalies). La règle : si la réplique qui suit rend l'explication redondante, couper l'explicative.

## Références

- Milan Kundera (La Plaisanterie, L'Insoutenable Légèreté de l'être — digressions, essai dans le roman)
- W. G. Sebald (Les Émigrants, Austerlitz — mélange texte/image, prose/document, fiction/essai)
- Jean-Philippe Toussaint (Fuir, Nue — l'hybridation discrète, le cinéma dans le roman)
- David Foster Wallace (Infinite Jest — l'hybridation totale, notes, digressions, changements de régime)
- Maggie Nelson (Les Argonautes — poésie/essai/récit, la forme qui épouse la pensée)

## À éviter dans ce style

- ✗ **Changement non signalé** : passer de la prose au vers sans transition → le lecteur est perdu
- ✗ **Hybridation décorative** : changer de genre parce que « c'est beau » → le changement doit servir le sens
- ✗ **Surenchère formelle** : un paragraphe en vers, le suivant en prose, puis en théâtre, puis en liste → le lecteur sature
- ✗ **Pacte rompu** : une séquence en prose réaliste suivie d'un passage fantastique sans justification → cohérence interne avant tout

# Index des skills

> Organisé par famille. Met à jour ce fichier quand un skill est ajouté, modifié ou retiré.
> Les skills sont répartis en trois catégories : **Formes** (contraintes structurelles), **Influences** (voix de départ), **Voix** (voix utilisateur générées).

---

## Formes — Contraintes structurelles et règles de genre

Ces skills décrivent les **invariants formels** : les règles du genre, la structure, les conventions. Ils sont neutres sur le plan esthétique — ils disent *comment c'est fait*, pas *quel style avoir*.

| Skill | Description | Statut |
|---|---|---|
| `ecriture-romanesque` | Conventions de mémoire, structure et process du projet romanesque | ✅ Actif |
| `ecriture-theatrale` | Écriture théâtrale contemporaine — dialogues, didascalies, sous-texte, rythme de scène | ✅ Actif |
| `ecriture-essai-litteraire` | Prose d'idées, mélange essai/récit, voix subjective | ✅ Actif |
| `ecriture-universitaire` | Mémoire, thèse, article — rigueur argumentative, clarté, citation | ✅ Actif |
| `ecriture-hybride` | Mélange roman/poésie/théâtre/essai dans un même texte | ✅ Actif |
| `ecriture-carnet-journal` | Fragments, notes, textes-mobiles, mémoire vive | ✅ Actif |
| `ecriture-epique` | Registre épique — souffle, accumulation, héros, fresque, mythe | ✅ Actif |
| `nouvelle-litteraire` | Nouvelle littéraire — concision, chute, intensité, économie de moyens | ✅ Actif |
| `flash-fiction` | Flash fiction — récit très court (<1000 mots), chute, intensité, économie maximale | ✅ Actif |
| `micro-nouvelle` | Micro-nouvelle — récit ultra-court (<300 mots), concentration poétique | ✅ Actif |
| `vignette-prose` | Vignette en prose — scène brève, atmosphérique, entre prose et poème | ✅ Actif |
| `composition-recueil` | Composition de recueil multi-textes — anaphore, écho, progression en croix, condensation | ✅ Actif |

---

## Influences — Voix de départ (starter packs esthétiques)

Ces skills sont des **propositions esthétiques** que l'utilisateur peut choisir comme point de départ. Ils ne sont pas activés par défaut. Ils incarnent une tradition, une école, un univers sonore — et peuvent être combinés ou abandonnés au profit d'une voix personnelle.

| Skill | Description | Statut |
|---|---|---|
| `poesie-contemporaine` | Vers libres, écriture moderne, influences actuelles | ✅ Actif |
| `poesie-classique` | Sonnets, alexandrins, formes fixes, tradition française | ✅ Actif |
| `poesie-prose` | Poème en prose — inspiration Rimbaud, Illuminations | ✅ Actif |
| `poesie-symbolique` | Poésie symbolique — Rimbaud, Verhaeren, Mallarmé, fin XIXe | ✅ Actif |
| `poesie-lyrique` | Poésie lyrique — émotion, chant, registre élégiaque | ✅ Actif |
| `poesie-engagee` | Poésie engagée — voix des opprimés, Artaud, Césaire | ✅ Actif |
| `poesie-madrigal-contemporain` | Madrigal contemporain — musicalité, lyrisme et chute, en vers libres | ✅ Actif |
| `roman-espionnage` | Roman d'espionnage réaliste — atmosphère Le Carré, méfiance, zones grises | ✅ Actif |
| `roman-romance` | Romance littéraire — relations complexes, non-dits, gestes minuscules, émotion retenue | ✅ Actif |
| `roman-litteraire` | Roman littéraire général — prose soignée, psychologie des personnages, ambiances | ✅ Actif |

---

## Voix — Skills générés par agent-style

Ces skills sont **créés automatiquement** par l'agent-style à partir des échantillons de l'utilisateur. Ils capturent sa signature personnelle. `voix-neutre` est la voix par défaut, activée quand aucune influence n'est sélectionnée.

| Skill | Description | Statut |
|---|---|---|
| `voix-neutre` | Aucun présupposé esthétique — la voix émerge du processus | ✅ Actif (défaut) |
| *(à générer)* | Voix personnelle créée à partir des échantillons utilisateur | ⏳ agent-style |

## Légende

- ✅ Actif — prêt à l'emploi
- ⏳ En création — en attente d'une action (agent-style, skill-manager)
- ❌ Obsolète — à archiver
